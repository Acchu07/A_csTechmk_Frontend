# CS Tech MK FrontEnd

### Setup

1. `git clone <repo>`
2. `cd into the repo`
3. `npm install`
4. `npm run dev`

### Notes

`Incase you change baseurlBackend make sure to change the baseURL in url.ts located in src/url.ts in the frontend`
`Incase you change the urlfrontend please update the same in the cors in the backend`
